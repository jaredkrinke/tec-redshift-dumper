TEC Redshift Dumper

This is a tool to dump TEC Redshift discs (from Exapunks) into a simple text format (and JSON). Drag and drop a disc (PNG image) onto the drop target below and content will appear in the corresponding text boxes.

All of the reverse-engineering comes from TEC Redshift Disk Specification by Rami-Sabbagh. This tool uses UPNG.js.

Source code for this tool is available via "view source" or this repository.
